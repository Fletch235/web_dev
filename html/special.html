A Special Time with Special Characters

Begin by finding the special character entities reference on W3Schools. You will need this to accomplish the job outlined here.

Here are all the lower case Greek letters and how to make them in a handy-dandy list. Notice that the<pre> tag has an inline cousin <code>. Use it where appropreiate!

  1. α is made using &alpha;
  2. β is made using &beta;
  3. γ is made using &gamma;
  4. δ is made using &delta;
  5. ε is made using &epsilon;
  6. ζ is made using &zeta;
  7. η is made using &eta;
  8. θ is made using &theta;
  9. ι is made using &iota;
  10. κ is made using &kappa;
  11. λ is made using &lambda;
  12. μ is made using &mu;
  13. ν is made using &nu;
  14. ξ is made using &xi;
  15. ο is made using &omicron;
  16. π is made using &pi;
  17. ρ is made using &rho;
  18. σ is made using &sigma;
  19. τ is made using &tau;
  20. υ is made using &upsilon;
  21. φ is made using &phi;
  22. χ is made using &chi;
  23. ω is made using &omega;
